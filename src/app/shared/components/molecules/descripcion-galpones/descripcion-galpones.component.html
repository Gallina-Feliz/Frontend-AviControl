Descripción de Galpones

Gallina Feliz

Un galpón es una estructura cerrada y espaciosa utilizada en la avicultura para albergar gallinas ponedoras, proporcionando un entorno controlado y seguro. Nuestro sistema integrará tecnología Arduino con sensores de peso y RFID para monitorear datos vitales de cada gallina, como raza, edad, producción diaria de huevos y estado de salud. Esta información será accesible para agricultores, enfermeros y administradores autorizados, facilitando la gestión y optimización de la producción de huevos y el cuidado de las gallinas.


Proceso de Gestión con el Sistema

Instalación de Tecnología:

1. Equipar el galpón con sensores de peso y etiquetas RFID conectadas a sistemas Arduino.

- Instalar lectores RFID y balanzas en las áreas donde las gallinas se alimentan y ponen huevos.


Registro y Monitoreo de Datos:

2. Asignar una etiqueta RFID a cada gallina para su identificación única.

- Los sensores de peso registran la producción diaria de huevos y las balanzas miden el peso de las gallinas.

- Los lectores RFID capturan la actividad y movimiento de cada gallina


Evaluación de la Salud:

3. Enfermeros autorizados realizan evaluaciones periódicas del estado de salud de las gallinas y registran la información en el sistema.

- Los datos de salud incluyen observaciones sobre el comportamiento, signos de enfermedad y tratamientos administrados.


Almacenamiento y Gestión de Datos:

4. Toda la información recopilada por los sensores y los enfermeros se almacena en una base de datos centralizada.

- Se utilizan procedimientos almacenados y transacciones para asegurar la integridad y consistencia de los datos


Acceso y Uso de la Información:

5.Agricultores, enfermeros y administradores autorizados acceden al sistema a través de una aplicación web desarrollada en ASP.NET Core.

La aplicación permite visualizar datos en tiempo real sobre la producción de huevos, estado de salud y otros parámetros críticos.


Toma de Decisiones:

6. Basado en la información recopilada, los agricultores pueden tomar decisiones informadas sobre el manejo de las gallinas, como ajustes en la alimentación o intervención veterinaria.

- Los administradores pueden optimizar la producción y mejorar las condiciones del galpón para maximizar el bienestar de las gallinas.

Beneficios del Sistema :

- Monitoreo Continuo: Provee datos en tiempo real sobre la producción y salud de cada gallina.

- Gestión Eficiente: Facilita la gestión y permite la identificación temprana de problemas de salud.

- Acceso Centralizado: Información accesible para todos los usuarios autorizados a través de una plataforma web.

- Optimización de la Producción: Permite ajustar las condiciones del galpón para mejorar la producción de huevos y el bienestar animal.
